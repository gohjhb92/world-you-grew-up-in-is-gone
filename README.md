# The World You Grew Up In Is Gone

An immersive interactive editorial web experience about the end of the liberal world order.

## Stack

- Next.js 15 App Router
- TypeScript
- Tailwind CSS
- Framer Motion
- Lucide Icons

## Local Development

```bash
npm install
npm run dev
```

## Static Build

```bash
npm run build
```

The project is configured with `output: "export"` so it can be deployed as a static site.
